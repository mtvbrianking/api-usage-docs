## Setup

```bash
$ npm install
```

```bash
$ npm run dev
```

## Serve

```bash
$ node app.js
```

## Usage

**ReDoc**

```bash
http://localhost:3000
```

**Swagger UI**

```bash
http://localhost:3000/swagger
```

**Swagger Editor**

```bash
http://localhost:3000/editor
```

**Swagger Live Edit**

Default spec: `public/specs/swagger.yaml`

```bash
http://localhost:3000/swagger/edit
```
